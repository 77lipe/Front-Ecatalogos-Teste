# Imagens dos Produtos

Para usar as fotos reais que você enviou, coloque os arquivos de imagem na pasta `public/images/` com os seguintes nomes:

## Produto 1 - Camiseta Branca (ID: 43)
- `camiseta-branca-1.jpg` - Vista frontal principal
- `camiseta-branca-2.jpg` - Vista lateral/detalhe  
- `camiseta-branca-3.jpg` - Vista costas

## Produto 2 - Camiseta Preta (ID: 44)
- `camiseta-preta-1.jpg` - Vista frontal principal
- `camiseta-preta-2.jpg` - Vista lateral/detalhe
- `camiseta-preta-3.jpg` - Vista costas

## Como funciona:
1. Copie as fotos que você enviou para a pasta `public/images/`
2. Renomeie os arquivos conforme a lista acima
3. A aplicação irá carregar automaticamente as imagens reais

## Estrutura esperada:
```
public/
└── images/
    ├── camiseta-branca-1.jpg
    ├── camiseta-branca-2.jpg
    ├── camiseta-branca-3.jpg
    ├── camiseta-preta-1.jpg
    ├── camiseta-preta-2.jpg
    └── camiseta-preta-3.jpg
```

As imagens serão usadas automaticamente na aplicação com navegação entre as 3 vistas de cada produto.
